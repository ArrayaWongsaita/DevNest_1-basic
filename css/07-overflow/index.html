1 visible (default)

content ล้นออกมาข้างนอก


.visible {
  overflow: visible;
}

2 hidden

content ล้นออกมาจะถูกตัดทิ้ง


.hidden {
  overflow: hidden;
}

3 scroll

แสดง scrollbar

overflow: scroll;

4 overflow-x and overflow-y

ความคุมแนวนอนแนวตั้งได้


overflow-x: visible;
overflow-y: scroll;

5 auto

แสดง scrollbar เมื่อจำเป็น


overflow: auto;
